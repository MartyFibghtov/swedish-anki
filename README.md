# swedish-anki
Anki add-on for easier creation of cards for swedish using https://svenska.se/ and https://folkets-lexikon.csc.kth.se/folkets/


# Desired card design:

## EN -> SV

### Front: 
- English form
- Image

### Back
- Swedish form with ett/en
- Examples 

### Features: 
- Play sound of turn

## SV -> EN
### Front: 
- Swedish word with ett en
- Play sound of turn 

### Back
- Definition in Swedish
- English translation
- Image
